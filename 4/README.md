# Прерывания BIOS

## Задание - Калькулятор суммы

Написать программу, принимающую на вход два слагаемых в виде hex цифр, считает сумму и выводит её не экран тоже, в hex виде.
Критерии оценки: Пример ввода:
Slag1 = 12345
Slag2 = 2
Summa: 12347
